🔚 마무리

이번 프로젝트를 통해 Node.js기반의 기능 구현 경험을 쌓을 수 있었으며,

미니게임과 친구 추가 기능을 개발하며 비동기 처리, 데이터 관리에 대한 이해도를 높일 수 있었습니다. 

특히 미니게임 개발 과정에서 게임 로직 설계 및 충돌 감지등을 경험할 수 있었다.

향후 기능 확장을 고려하여 코드 구조를 개선할 계획입니다.
